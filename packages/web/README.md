# @clonelab/web

CloneLab web application - Interactive molecular cloning toolkit.

## Development

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev

# Build for production
pnpm build

# Preview production build
pnpm preview
```

## Features

- Upload and visualize DNA sequences
- Find restriction enzyme sites
- Simulate digests with virtual gel electrophoresis
- Plan ligation strategies
- Export results

## Tech Stack

- React 18
- Redux Toolkit
- Vite
- Tailwind CSS
- TypeScript
