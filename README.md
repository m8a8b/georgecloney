# CloneLab

A modern molecular cloning toolkit for planning and visualizing restriction digests and ligations.

## Project Structure

This is a monorepo containing:

- **@clonelab/core** - Framework-agnostic TypeScript library for molecular biology operations
- **@clonelab/web** - React frontend application with interactive visualizations

## Getting Started

```bash
# Install dependencies
pnpm install

# Run development server
pnpm dev

# Build all packages
pnpm build

# Run tests
pnpm test
```
